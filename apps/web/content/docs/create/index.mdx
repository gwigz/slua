`@gwigz/slua-create` is an interactive CLI that scaffolds a ready-to-build
SLua TypeScript project in seconds. It walks you through template selection,
optional extras, and initial configuration.

## Quick Start

```bash tab="npm"
npx @gwigz/slua-create
```

```bash tab="pnpm"
pnpm dlx @gwigz/slua-create
```

```bash tab="bun"
bunx @gwigz/slua-create
```

You can also pass a directory name as a positional argument to skip the
directory prompt:

```bash
npx @gwigz/slua-create my-slua-project
```
