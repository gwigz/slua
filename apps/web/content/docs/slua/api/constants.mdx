The SLua runtime defines over 1,000 constants. Below are the most commonly
used groups. All constants are available as globals, no import needed.

## Common Constants

| Constant        | Type         | Description                   |
| --------------- | ------------ | ----------------------------- |
| `ZERO_VECTOR`   | `vector`     | `<0, 0, 0>`                   |
| `ZERO_ROTATION` | `quaternion` | `<0, 0, 0, 1>`                |
| `NULL_KEY`      | `uuid`       | Null UUID                     |
| `PI`            | `number`     | 3.14159...                    |
| `TWO_PI`        | `number`     | 6.28318...                    |
| `DEG_TO_RAD`    | `number`     | Degrees to radians multiplier |
| `RAD_TO_DEG`    | `number`     | Radians to degrees multiplier |
