# `@gwigz/slua-modules`

Shared runtime modules for [SLua](https://github.com/gwigz/slua) projects.

## Installation

```sh
npm install @gwigz/slua-modules
# or
pnpm add @gwigz/slua-modules
# or
bun add @gwigz/slua-modules
```

## Modules

| Module                        | Description                                                        |
| ----------------------------- | ------------------------------------------------------------------ |
| `@gwigz/slua-modules/config`  | Typed notecard config with YAML and lljson parsers                 |
| `@gwigz/slua-modules/yield`   | Coroutine wrappers that flatten callback APIs into sequential code |
| `@gwigz/slua-modules/testing` | Mock utilities for unit testing SLua modules                       |

## Documentation

Full API reference and usage examples are available at [slua.gwigz.link/docs/modules](https://slua.gwigz.link/docs/modules).
